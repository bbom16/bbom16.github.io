# django 설치하기
-  #### pycharm 사용


1. pycharm으로 프로젝트 생성 후 terminal 들어가기

2. terminal에 (venv) 표시가 없다면 아래 코드 terminal에 입력
```
$ source venv/bin/activate
```

3. django 설치하기!!
```
(myvenv) ~$ pip install django~=1.11.0
```

참고사항 : [장고걸스](https://tutorial.djangogirls.org/ko/django_installation/)
